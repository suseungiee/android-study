# git에 md파일 올릴때 요령

이미지 추가하면 이름이 Untitled, Untitled1, Untitled2... 이렇게 되므로

이미지 넣고 그위에

느낌표["파일이름"](./폴더이름/"파일이름.확장자")이렇게 해주면 git에서 이미지 바로보인다
이러면 굳이 issue로 이미지올려서 일일히 편집 안해줘도된다.

여기서 폴더이름이란 내보내기해서 이미지 저장되는 폴더이름
